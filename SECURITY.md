# Security Policy

## Supported Versions

The following versions of **Flaming Cherubim** are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of our software seriously. If you believe you have found a security vulnerability in **Flaming Cherubim**, please report it to us as described below.

**Please do not report security vulnerabilities through public GitHub issues.**

### How to Report

Please report security vulnerabilities by emailing the maintainer directly or reaching out via the contact information on [danials.org](https://danials.org).

Please include:
*   A description of the vulnerability.
*   Steps to reproduce the issue.
*   Any relevant logs or screenshots.

We will acknowledge your report and do our best to resolve the issue as quickly as possible.
